# Credentials Reference — PLACEHOLDERS ONLY
# NEVER commit real keys, IDs, or wallet addresses here.
# Set all values in Render dashboard (sync: false) or via environment only.

## Required env vars (set in Render dashboard)

```
PAY_TO_WALLET=<your-Base-USDC-wallet-address>         # Base chain, checksummed EIP-55
SOLANA_PAY_TO=<your-Solana-wallet-pubkey>              # 32-byte base58 pubkey
CDP_API_KEY_ID=<your-CDP-key-id>                       # Coinbase Developer Platform key ID
CDP_API_KEY_SECRET=<your-CDP-key-secret>               # Never log or commit
UPSTASH_REDIS_REST_URL=<from console.upstash.com>      # Create DB → REST tab
UPSTASH_REDIS_REST_TOKEN=<from console.upstash.com>
RESPONSE_SIGNING_KEY=<generate: node sign.js --gen>    # Ed25519 private key
```

## Auto-generated by Render (do not set manually)

```
TOKEN_SECRET   # generateValue: true in render.yaml
```

## Notes
- CDP keys unlock mainnet Base settlement AND the Solana (/sol/*) routes.
- Base + Polygon work with PAY_TO_WALLET alone; Solana needs CDP keys present.
- Without Upstash, loyalty/affiliate ledger runs in-memory (resets on restart).
- RESPONSE_SIGNING_KEY is generated once per deployment; back it up securely.

## Security
- Rotate CDP keys immediately if you believe they were exposed.
- All secrets must be set via Render environment variables only, never in source.
- See render.yaml for the full list of required env var names (all `sync: false`).
